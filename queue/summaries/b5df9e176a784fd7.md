GitHub Copilot in JetBrains IDEs, Xcode, and Eclipse now features an "Auto" option for model selection. This option automatically chooses the best Copilot model for you. You can select "Auto" from the model picker in Copilot Chat.

For more details, see the full announcement on the [GitHub Blog](https://github.blog/changelog/2025-11-18-auto-model-selection-for-copilot-in-jetbrains-ides-xcode-and-eclipse-in-public-preview).
