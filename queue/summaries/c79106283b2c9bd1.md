GitHub Copilot now offers custom agents in public preview for JetBrains IDEs, Eclipse, and Xcode. These custom agents let you package reusable expertise, set guardrails, and provide tailored tool access, creating AI assistants specifically customized for your projects.

For more details, see the [GitHub Blog announcement](https://github.blog/changelog/2025-11-18-custom-agents-available-in-github-copilot-for-jetbrains-eclipse-and-xcode-now-in-public-preview).
