The recent CodeQL 2.23.5 release includes support for Swift 6.2, three new Java security queries, and improved analysis accuracy. CodeQL powers GitHub's code scanning to help find and fix security issues in codebases.

For more details, you can read the full announcement on the [GitHub Blog](https://github.blog/changelog/2025-11-19-codeql-2-23-5-adds-support-for-swift-6-2-new-java-queries-and-improved-analysis-accuracy).
